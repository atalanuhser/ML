# ML Math Image Process (C#)

Bu proje, C# ve Visual Studio kullanılarak geliştirilmiş basit bir görüntü işleme uygulamasıdır. Amaç, seçilen bir görsel üzerinde temel işlemler uygulamak ve sonucu kullanıcıya sunmaktır.

## Özellikler
- Bilgisayardan görsel seçme
- Görsel üzerinde temel görüntü işleme işlemleri
- İşlenmiş görüntüyü ekranda görüntüleme

## Kullanılan Teknolojiler
- C# (.NET)
- Visual Studio
- Windows Forms / WPF

## Kurulum ve Çalıştırma
1. Bu depoyu bilgisayarınıza klonlayın:
   ```bash
   git clone <repo-link>
2.Projeyi Visual Studio ile açın (.sln dosyası).

3.Gerekli paketler varsa NuGet üzerinden restore edin.

4.Projeyi çalıştırın.

Kullanım

1.Uygulamayı başlatın.

2.Görsel seçme butonu ile bir resim yükleyin.

3.Uygulamak istediğiniz işlemi seçin.

4.Sonucu ekranda görüntüleyin.
